# 👤 Profile Card

A simple and stylish profile card built with **HTML**, **CSS**, and **JavaScript**. This project showcases a personal profile with key information, social links, and a live UTC time display.

## 📌 Project Description

This profile card includes:

- A profile picture
- Full name and job title
- A short bio
- Location and email address
- Links to GitHub and LinkedIn
- Real-time UTC clock display

All elements are styled for a clean and modern look using custom CSS and Font Awesome icons.

## 🧩 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- [Font Awesome](https://fontawesome.com/) for icons
